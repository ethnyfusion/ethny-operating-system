# Charte graphique finale 2026

Source de reference pour le site web Ethny et les documents internes de l'Ethny Operating System.

## Sources officielles

- PDF : `Ethny-Nomad-Cuisine-Charte-Graphique-2026 (1).pdf`
- Design system : `Ethny Nomad Cuisine Design System/`
- Logos finaux : `ethny-logo-A-final-figma-svg/`

## Positionnement

Ethny Nomad Cuisine est une marque premium, chaleureuse, humaine, claire et rassurante.

Formulation de reference :

> Premium sans arrogance. Chaleureux sans familiarite excessive. Clair, concret, rassurant.

La marque porte une experience culinaire sur mesure : base francaise, influences Bresil, Asie, Mediterranee, Orient et cuisine nomade, avec une presence du chef precise, discrete et orientee service.

## Palette officielle

- Foret profond : `#1C3B34`
- Vert foret : `#285447`
- Sauge : `#5E8D7A`
- Creme : `#F2EBDC`
- Pierre : `#C9C5BB`
- Anthracite : `#2A2A2A`
- Blanc surface : `#FFFFFE`

Ratio recommande :

- Creme dominante : environ 60 %
- Vert foret structurant : environ 25 %
- Sauge en respiration : environ 10 %
- Pierre pour lignes et fonds secondaires : environ 5 %

Ne pas ajouter de rouge, orange, jaune vif, bleu ou couleurs promotionnelles hors palette.

## Typographies

- Titres, logo, citations : `Cormorant Garamond`
- Paragraphes, boutons, formulaires, informations pratiques : `Jost`

Les textes doivent rester lisibles, calmes et premium. Les capitales espacees sont reservees aux labels courts, sourcils de section et signatures.

## Formes et composants

- Badges : coins tres legerement arrondis, `3px`
- Petits pictos / elements compacts : `6px`
- Boutons : coins soft, `10px`
- Cartes : `14px`
- Ombres : tres douces, peu contrastees
- Bordures : hairline, pierre ou vert foret selon le contexte

Eviter les boutons ou badges en pilule. Eviter les ombres lourdes, gradients decoratifs et effets SaaS generiques.

## Logos

Le logo principal est le sceau cercle-brise avec le mot-symbole `ethny` et la signature `NOMAD CUISINE`.

Regles :

- conserver une zone de respiration autour du logo ;
- ne pas deformer, incliner, ombrer ou entourer le logo ;
- ne pas ajouter de couleur vive ou effet brillant ;
- utiliser la version claire sur fond foret et la version foncee sur fond clair.

## Iconographie

Utiliser les pictogrammes lineaires de la charte finale :

- trait fin ;
- monochrome via la couleur courante ;
- pas d'emoji ;
- pas de symbole Unicode improvise ;
- pas de pictogramme invente hors famille graphique.

## Photographie

Priorite aux photos reelles :

- plats ;
- dressages ;
- mains en cuisine ;
- service ;
- ingredients ;
- tables et lieux seulement en soutien.

Le rendu doit etre naturel, doux, side-lit, premium et humain. Eviter l'effet stock, les couleurs saturees et les images trop decoratives.

## Ton editorial

Toujours distinguer :

- estimation ;
- proposition ;
- devis confirme.

Ne jamais promettre automatiquement une reservation, un prix final ou une disponibilite.

CTA recommandes :

- Demander une proposition
- Verifier la disponibilite
- Recevoir un devis personnalise
- Demander un devis
