# Conventions de nommage Canva

## Structure générale

Format recommandé :

```text
ethny-[canal]-[type]-[sujet]-[format]-[version]
```

Exemple :

```text
ethny-instagram-carousel-chef-prive-1080x1350-v01
```

## Règles

- tout en minuscules ;
- pas d'accent dans les noms de fichiers exportés ;
- mots séparés par des tirets ;
- inclure le canal ;
- inclure le format ;
- versionner avec `v01`, `v02`, `v03` ;
- ajouter `approved` uniquement après validation humaine.

## Préfixes

| Préfixe | Usage |
| --- | --- |
| `ethny-logo-` | Logos officiels |
| `ethny-icon-` | Icônes et favicons |
| `ethny-picto-` | Pictogrammes de charte |
| `ethny-bg-` | Arrière-plans |
| `ethny-texture-` | Textures |
| `ethny-template-` | Modèles Canva |
| `ethny-photo-` | Photos |
| `ethny-video-` | Vidéos |

## Exemples

```text
ethny-logo-principal-cercle-brise.svg
ethny-logo-blanc-fond-vert.svg
ethny-template-instagram-post-premium-1080x1350-v01
ethny-template-gbp-chef-a-domicile-1200x628-v01
ethny-template-menu-degustation-a4-v01
ethny-photo-dressage-pavlova-2026-v01
```

## Interdits

- `final-final`
- `test`
- `nouveau`
- `copie`
- `sans titre`
- noms basés sur des émotions : `joli`, `beau`, `wow`

Un fichier non nommé correctement n'est pas prêt à être partagé à l'équipe.
