# Maquettes Canva - Ethny Fusion

Ces maquettes décrivent les modèles à créer dans Canva. Elles ne remplacent pas les fichiers Canva eux-mêmes : elles servent de cahier de fabrication pour créer des modèles cohérents, rapides à utiliser et conformes à la charte.

## Règles communes

- Utiliser uniquement les logos officiels du dossier `ETHNY/Logos`.
- Utiliser uniquement les pictogrammes du dossier `ETHNY/Pictogrammes`.
- Couleur héro : `#1C3B34`.
- Fond clair prioritaire : `#F2EBDC`.
- Bordures et filets : `#C9C5BB`.
- Pas de pictogrammes Canva génériques.
- Pas de couleurs hors charte.
- Pas de promesse de disponibilité, prix final ou réservation.
- Photos réelles en priorité.

## Instagram

### Post Premium

- Format : 1080x1350.
- Structure : photo réelle plein cadre ou fond vert profond, titre court, logo discret, CTA.
- Usage : annonce service, événement, lancement.

### Carrousel

- Format : 1080x1350, 5 à 7 pages.
- Pages : couverture, contexte, détail, preuve, CTA.
- Règle : une idée par slide, texte court.

### Avant / Après

- Format : carrousel 2 à 4 pages.
- Pages : avant, détail de préparation, après, CTA.
- Attention : éviter de montrer clients sans accord.

### Témoignage client

- Format : 1080x1350.
- Structure : citation validée, nom ou initiales si autorisées, type d'expérience, logo discret.
- Interdit : inventer un avis.

### Citation

- Format : 1080x1350.
- Structure : fond crème, citation en serif, pictogramme officiel discret.
- Usage : marque, philosophie, exigence.

### Plat du jour

- Format : 1080x1350.
- Structure : photo plat, titre, 2 à 3 ingrédients, CTA doux.
- Règle : pas de prix si non validé.

### Signature Dish

- Format : 1080x1350.
- Structure : photo forte, nom du plat, inspiration, détail sensoriel sobre.
- Usage : pavlova, moqueca, ravioles, picanha, aubergine miso.

### Événement

- Format : 1080x1350.
- Structure : type d'événement, ambiance, détail service, CTA.
- Règle : pas de lieu privé identifiable sans accord.

### Making Of

- Format : reel cover 1080x1920.
- Structure : geste chef, titre court, pictogramme discret.

### Coulisses

- Format : story ou reel cover.
- Structure : plan vertical, texte court, logo discret.

### Astuce du Chef

- Format : carrousel 4 à 6 slides.
- Pages : astuce, pourquoi, comment, erreur à éviter, CTA.

### Recette

- Format : carrousel 5 à 8 slides.
- Pages : photo, ingrédients, étapes, dressage, conseil.

### Produit local

- Format : 1080x1350.
- Structure : photo produit, origine validée, usage culinaire.

### Saison

- Format : 1080x1350.
- Structure : fond crème, photo ingrédient, saison, idée menu.

### FAQ

- Format : carrousel 5 slides.
- Pages : question, réponse courte, précision, CTA.

### Promotion

- Format : 1080x1350.
- Usage rare.
- Règle : ne jamais donner une impression low-cost.

### Story

- Format : 1080x1920.
- Structure : 1 idée, 1 photo ou fond, texte court.

### Story Question

- Format : 1080x1920.
- Zone libre pour sticker question Instagram.

### Story Sondage

- Format : 1080x1920.
- Zone libre pour sticker sondage.

### Story Countdown

- Format : 1080x1920.
- Zone libre pour sticker compte à rebours.

### Reel Cover

- Format : 1080x1920.
- Structure : sujet lisible au centre, titre court, logo discret.

## Google Business Profile

Format commun : 1200x628.

Templates :

- Actualité : annonce courte + photo.
- Nouvel avis : citation validée + logo.
- Nouveau menu : nom menu + 3 lignes descriptives.
- Chef à domicile : service + photo table ou chef.
- Traiteur : événement + bouchées / service.
- Mariage : ambiance sobre, jamais cliché.
- Cours de cuisine : geste pédagogique.
- Promotion : à utiliser rarement, sans ton discount.
- Horaires : information claire, peu décorée.
- Nouvel événement : type événement + CTA.

CTA recommandés :

- Demander une proposition
- Vérifier la disponibilité
- Envoyer une demande

## Menus

Format commun : A4 portrait.

Templates :

- Menu dégustation
- Menu mariage
- Menu entreprise
- Menu privé
- Buffet
- Brunch
- BBQ
- Cocktail dînatoire

Structure :

1. logo officiel ;
2. nom du menu ;
3. contexte ou saison ;
4. services ;
5. options ;
6. notes allergènes ;
7. signature.

Règle : ne pas afficher un prix final si les variables ne sont pas validées.

## Fiche recette professionnelle

Format : A4 portrait.

Blocs obligatoires :

- Photo
- Difficulté
- Temps
- Portions
- Ingrédients
- Matériel
- Préparation
- Dressage
- Conseils du chef
- Allergènes

Règles :

- lisible en impression ;
- hiérarchie claire ;
- notes allergènes visibles ;
- pas de surcharge décorative.
