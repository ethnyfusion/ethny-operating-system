# Canva System - Ethny Fusion

Ce dossier prépare l'importation structurée d'Ethny Fusion / Ethny Nomad Cuisine dans Canva.

Objectif : permettre à l'équipe de créer un visuel professionnel en moins de 5 minutes, sans repartir d'une page blanche et sans s'éloigner de la charte graphique officielle.

## Source unique

La référence graphique obligatoire reste :

- `ethny-operating-system/05_content_branding/brand-system/charte-graphique-finale-2026.md`
- `ethny/branding/logo/`
- `ethny/branding/icons/`
- `ethny/branding/colors.json`
- `ethny/branding/typography.md`

Toute génération Canva ou IA qui invente un logo, un pictogramme, un cercle brisé ou une palette est considérée comme non conforme.

## Structure

- `ETHNY/Brand Kit/` : fichiers à importer dans le Kit de marque Canva.
- `ETHNY/Logos/` : copies contrôlées des logos officiels.
- `ETHNY/Couleurs/` : palette officielle et documentation HEX.
- `ETHNY/Typographies/` : hiérarchie typographique.
- `ETHNY/Pictogrammes/` : pictogrammes SVG officiels.
- `ETHNY/Icônes/` : icônes officielles, favicons et usages.
- `ETHNY/Templates .../` : emplacements des modèles Canva.
- `ETHNY/Photos/` et `ETHNY/Vidéos/` : structure média à alimenter après tri et validation.

## Documents clés

- `importer-ethny-dans-canva.md` : guide d'importation et de configuration Canva.
- `folder-map.md` : arborescence à recréer dans Canva.
- `naming-conventions.md` : règles de nommage.
- `asset-manifest.csv` : inventaire des assets officiels préparés.
- `template-index.csv` : liste des modèles à créer.
- `template-blueprints.md` : maquettes fonctionnelles des templates.
- `media-library-manifest.csv` : structure de tri média.

## Statut

Préparation locale prête pour import manuel dans Canva Pro.

Limite actuelle : les outils disponibles ne permettent pas d'importer directement des fichiers SVG locaux dans Canva comme assets de bibliothèque. Les fichiers sont donc préparés dans le repo avec la documentation nécessaire pour un import Canva propre et contrôlé.
