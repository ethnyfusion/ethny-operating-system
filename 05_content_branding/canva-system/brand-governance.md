# Gouvernance Canva - Ethny

## Principe

Canva est un outil de production, pas une source de vérité graphique.

La source de vérité reste :

- la charte graphique officielle ;
- les SVG officiels ;
- les couleurs documentées ;
- les règles typographiques ;
- les validations humaines.

## Ce que Canva peut faire

- accélérer la production ;
- dupliquer des modèles ;
- adapter un format ;
- préparer des posts ;
- ranger les assets ;
- partager les modèles à l'équipe.

## Ce que Canva ne doit pas faire

- inventer un logo ;
- inventer un pictogramme ;
- fermer ou modifier le cercle brisé ;
- créer une nouvelle palette ;
- remplacer la validation humaine ;
- générer des visuels publiables sans contrôle.

## Processus de validation

Un design est publiable uniquement si :

1. le logo vient du dossier officiel ;
2. les pictogrammes viennent du dossier officiel ;
3. les couleurs sont dans la palette officielle ;
4. la typographie suit la hiérarchie ;
5. le texte ne contient pas de promesse risquée ;
6. Chef Reginald ou la personne responsable valide le rendu final.

## Statuts recommandés

- `draft` : brouillon de travail.
- `review` : à relire.
- `approved` : validé.
- `published` : publié.
- `archive` : ancien modèle ou ancienne version.

## Règle simple

Si un élément visuel n'existe pas dans le repo ou dans le Brand Kit validé, il ne doit pas être utilisé comme élément de marque.
