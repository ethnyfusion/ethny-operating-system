# Guide upload Canva

## Méthode recommandée

Dans Canva :

1. Ouvrir le dossier `ETHNY`.
2. Créer ou ouvrir le sous-dossier correspondant.
3. Cliquer sur `Importer`.
4. Sélectionner les fichiers `.svg` du dossier local équivalent.
5. Vérifier visuellement chaque fichier importé.
6. Ranger les fichiers dans le bon dossier Canva.

## Brand Kit

Dans le Kit de marque Canva :

1. Ajouter les logos depuis `01_logos`.
2. Ajouter les couleurs manuellement avec les codes HEX.
3. Configurer les polices :
   - Titres : Cormorant Garamond.
   - Textes : Jost.
4. Ne pas créer de logo ou pictogramme avec Canva AI.

## À ne pas faire

- Ne pas remplacer les logos officiels par une génération Canva.
- Ne pas utiliser de pictogrammes Canva génériques.
- Ne pas utiliser le dossier `canva-upload-svg-pack-rejected-generated`.
- Ne pas ajouter d'effets au logo.
- Ne pas fermer le cercle brisé.
- Ne pas changer la palette.

## Contrôle rapide

Un fichier est prêt si :

- il est en `.svg` ;
- il porte un nom `ethny-...` ;
- il respecte la palette ;
- il peut être agrandi sans perte ;
- il ne contient pas d'élément hors charte.
