# 04 Finance

## Objectif

Centraliser les elements financiers utiles a la gestion d'Ethny : plan financier, tresorerie, hypotheses, suivi des revenus et protection de marge. Ce dossier doit aider a prendre des decisions claires sans inventer de chiffres non valides.

## Types de fichiers attendus

- Plans financiers.
- Suivis de tresorerie.
- Hypotheses de chiffre d'affaires.
- Grilles de couts.
- Notes de marge.
- Syntheses mensuelles.

## Prochaines actions possibles

- Creer un tableau simple de suivi tresorerie.
- Creer une grille de rentabilite par type de prestation.
- Documenter les hypotheses de prix et de couts.
- Ajouter une revue mensuelle : revenus, charges, marge, priorites.
