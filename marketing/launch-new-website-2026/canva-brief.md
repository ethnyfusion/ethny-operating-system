# Brief Canva - Lancement nouveau site 2026

## Direction artistique

Créer une série de visuels premium, chaleureux, artisanaux et contemporains autour de la nouvelle identité Ethny.

L'univers doit rester sobre, gastronomique et végétal. Le vert profond est la couleur héro. Les textures et le filigrane de vague servent de signature discrète, jamais de décor trop chargé.

## Palette

- Foret profond : `#1C3B34`
- Vert foret : `#285447`
- Sauge : `#5E8D7A`
- Creme : `#F2EBDC`
- Pierre : `#C9C5BB`
- Anthracite : `#2A2A2A`
- Blanc surface : `#FFFFFE`

Ratio conseillé :

- 60 % crème ou surface claire ;
- 25 % vert profond / vert forêt ;
- 10 % sauge ;
- 5 % pierre pour filets, séparateurs et détails.

## Typographies

- Titres : Cormorant Garamond, ou une serif élégante proche si non disponible.
- Textes courts : Jost, ou une sans serif claire et contemporaine si non disponible.

Règles :

- phrases courtes ;
- capitales espacées uniquement pour labels courts ;
- pas de typographie fantaisie ;
- lisibilité prioritaire en story et mobile.

## Templates carrousel

### Template 1 - Teasing détail

Format : 1080x1350.

Structure :

- slide 1 : détail visuel très cadré, phrase courte ;
- slide 2 : texture ou pictogramme ;
- slide 3 : fragment de logo ou couleur ;
- slide 4 : phrase de transition ;
- slide 5 optionnelle : CTA doux.

Exemples de textes :

- Une nouvelle étape se prépare.
- Un détail, une direction, une exigence.
- Nouvelle identité, même exigence.

### Template 2 - Service

Format : 1080x1350.

Structure :

- slide 1 : nom du service ;
- slide 2 : bénéfice client ;
- slide 3 : expérience culinaire ;
- slide 4 : détail de service ;
- slide 5 : CTA.

Services à décliner :

- chef à domicile ;
- chef privé ;
- traiteur événementiel ;
- mariage intime ;
- anniversaire premium ;
- cours de cuisine ;
- expérience privée.

### Template 3 - Preuve sociale

Format : 1080x1350.

Structure :

- citation validée ;
- photo réelle en arrière-plan doux ;
- logo discret ;
- mention du type d'expérience, sans données privées.

Ne jamais inventer d'avis. Utiliser uniquement un avis validé.

## Templates story

Format : 1080x1920.

Créer 5 modèles :

1. Compte à rebours Jour J.
2. Détail logo / texture.
3. Question sticker : "Quel détail voulez-vous découvrir ?"
4. Coulisses du site.
5. CTA après reveal : "Découvrir le nouveau site".

Règles :

- 1 idée par story ;
- texte très court ;
- zone de sécurité haute et basse pour l'interface Instagram ;
- logo discret, jamais collé aux bords.

## Templates reel

Format : 1080x1920.

Créer :

- cover reveal nouveau site ;
- cover geste du chef ;
- cover service chef privé ;
- cover traiteur événementiel ;
- cover cours de cuisine.

Rythme visuel :

- plans courts ;
- texture ou logo en intro 0,5 à 1 seconde ;
- plat, geste ou site en sujet principal ;
- CTA final sobre.

## Utilisation du logo

- Utiliser la version claire sur fond vert profond.
- Utiliser la version foncée sur fond crème.
- Conserver une zone de respiration visible.
- Ne pas étirer, ombrer, incliner ou encadrer le logo.
- Le logo complet doit être réservé au Jour J et aux posts post-lancement.
- Avant Jour J, utiliser seulement des fragments : cercle brisé, détail typographique, pictogramme.

## Utilisation des pictogrammes

- Monochrome uniquement.
- Couleur : crème sur fond vert ou vert profond sur fond crème.
- Taille discrète : signature, repère ou ponctuation.
- Ne pas utiliser comme emoji ou décoration répétitive.
- Garder la même épaisseur de trait.

## Fonds verts, texture et filigrane

Créer une bibliothèque Canva avec :

- fond vert profond uni ;
- fond vert avec texture papier subtile ;
- fond crème avec filigrane vague en 5 à 8 % d'opacité ;
- fond photo avec overlay vert transparent ;
- fond sauge très doux pour respirations.

Le filigrane de vague doit rester secondaire. Il doit donner une sensation de mouvement et de voyage, sans rendre le texte moins lisible.

## Règles de contraste

- Texte crème sur vert profond : recommandé.
- Texte vert profond sur crème : recommandé.
- Sauge sur crème : uniquement pour détails ou sous-titres courts.
- Éviter texte sauge sur vert.
- Éviter texte sur photo sans overlay.
- Vérifier chaque story sur mobile avant publication.
